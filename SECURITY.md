# Security Policy

## Supported Versions
Only version `1.0.0` is currently supported with active security updates.

## Reporting a Vulnerability
We take the security of MoodFlix seriously. If you find a security vulnerability, please do NOT log it as a public GitHub issue. Instead, report it privately:

1. Send an email to security@moodflix.com detailing the issue.
2. Include reproduction logs, code snippets, or configuration vectors.
3. We will acknowledge your report within 48 hours and work on a patch within 7 days.
