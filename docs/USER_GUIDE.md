# User Guide

Welcome to MoodFlix! This guide walks you through navigating the application, tracking your moods, and getting cinema recommendations.

---

## 1. Authentication
- **Registration**: Register an account with a unique email address.
- **Login**: Input your email and password to log in. Check "Remember Me" to save credentials across sessions.

---

## 2. Navigating the Dashboard
The main dashboard is divided into three key panels:
1. **Sidebar Menu**: Access your User Profile, Watchlist, Activity History, and System Settings. Click the collapse button (☰) to expand/minimize the layout.
2. **Interactive Search**: Use the search input box at the top to search the database. Debounced results show up instantly.
3. **Mood Discovery Panel**:
   - Select your current emotion from the dropdown (e.g., Happy, Calm, Sad).
   - Select the target content format (e.g., Movie, Series, Song).
   - Click **Recommend Matches** to generate custom, prioritized recommendations.

---

## 3. Keyboard Shortcuts
Navigate quickly using standard hotkeys:
- `Ctrl + S`: Focuses cursor on Search input
- `Ctrl + W`: Navigates to Watchlist
- `Ctrl + P`: Navigates to User Profile
- `Ctrl + H`: Navigates back to Home/Dashboard
- `Ctrl + L`: Log out of the application
